# Test Sphinx Site for pyodide Directive

This directory is a minimal Sphinx project used as a `testroot` by the `sphinx.testing` fixtures in `tests/test_build.py`.

`@pytest.mark.sphinx("html", testroot="pyodide")` tells pytest to build a Sphinx HTML site
using this directory as the source. The output is then inspected for correct HTML structure,
assets, and content.

### Files

- **conf.py** — Enables the `sphinx_pyodide` extension.
- **index.rst** — Contains two pyodide blocks:
    - A basic block (`print("hello")`)
    - A block with a package dependency (`numpy`)

### Usage

These tests do not require a running browser — they only verify the static HTML output
generated by Sphinx during a build.
